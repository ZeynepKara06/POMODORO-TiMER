# POMODORO TİMER
A simple Pomodoro timer built with python to improve productivity and practice programming skills.

## About
This Python program helps users manage work and break intervals using the classic pomodoro technique. This project was created to practice 
python programming,loops,functions, and simple time management logic in a fun and interactive way.

## Features
1. Takes the total work time input from the user
2. Converts the time into seconds and displays a live coutdown timer.
3. Divides the total time into 25 minutes work sessions and 5 minute breaks in a loop.
4. Displays a motivational message like "well, done!" when the timer finishes.
